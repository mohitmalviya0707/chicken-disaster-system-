# chicken-disaster-system-
# 🐔 Chicken Disease Detection System

A deep learning-based system that detects chicken diseases from fecal images.
The model classifies samples into multiple health conditions to assist in early diagnosis and poultry management.

---

## 📌 Project Overview

This project uses deep learning to analyze chicken fecal images and identify potential diseases.
It helps in quick and automated detection, reducing manual effort and improving farm productivity.

---

## 🧠 Classes Detected

| Class                 | Description                           |
| --------------------- | ------------------------------------- |
| 🦠 Coccidiosis        | Parasitic disease caused by *Eimeria* |
| ✅ Healthy             | Normal healthy chicken                |
| 🦠 New Castle Disease | Viral disease (NDV)                   |
| 🦠 Salmonella         | Bacterial infection                   |

---

## 🚀 Features

* Image-based disease detection
* Multi-class classification
* Fast and automated predictions
* Useful for poultry health monitoring
